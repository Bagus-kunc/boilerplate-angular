# 🚀 AngularScale - Premium Angular 2026 Boilerplate

A high-performance, future-ready Angular boilerplate designed for modern web applications. Built with the latest 2026 web standards.

## ✨ High-Performance Features

- **Angular 21+**: Fully leverages standalone components and advanced runtime optimizations.
- **Zoneless Architecture**: Drastically reduces initial bundle sizes and improves change detection performance by removing `zone.js`.
- **Angular Signals**: Modern, re-active state management as a first-class citizen.
- **Tailwind CSS 4**: Next-generation utility-first styling for maximum development speed and performance.
- **SSR & Hydration**: Server-side rendering and client-side hydration pre-configured for optimal SEO and performance.

## 🛠️ Stack

- **Framework**: [Angular 21+](https://angular.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons**: [Lucide Angular](https://lucide.dev/)
- **State Mgmt**: [Angular Signals](https://angular.dev/guide/signals)
- **Tooling**: [Vite](https://vitejs.dev/) & [Vitest](https://vitest.dev/)

## 🚀 Getting Started

### 1. Install Dependencies

```bash
cd boilerplate-angular
npm install
```

### 2. Run Development Server

```bash
npm start
```

Open [http://localhost:4200](http://localhost:4200) with your browser to see the result.

## 📁 Project Structure

- `src/app/`: Core application logic (Signals, Services, Components).
- `src/styles.css`: Global design system and Tailwind 4 integration.
- `public/`: Static assets and icons.

## 🏗️ Building for Production

```bash
npm run build
```

The output will be in the `dist/boilerplate-angular` directory.
